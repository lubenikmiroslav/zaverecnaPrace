# Font Howdybun

Pro správné zobrazení fontu Howdybun v aplikaci:

1. Stáhni font z: https://www.1001fonts.com/howdybun-font.html
2. Zkopíruj stažený soubor `Howdybun-Regular.ttf` (nebo podobný název) do této složky
3. Ujisti se, že soubor se jmenuje přesně `Howdybun-Regular.ttf`
4. Spusť `flutter pub get` v kořenovém adresáři projektu

Font se použije pouze pro nápis "HabitTrack" na hlavní obrazovce.
